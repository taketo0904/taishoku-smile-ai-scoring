# 退職スマイル 商談AIスコアリング

商談（着席〜クロージング）の文字起こしをAIに読ませ、6カテゴリ・100点満点のルーブリックで自動採点するツールです。

- 冒頭設計・主導権（10点）
- ヒアリング深度（25点・最重要）
- プレゼン・価値訴求（20点）
- 不安解消・切り返し（15点）
- クロージング（20点）
- コンプラ・期待値管理（10点・減点方式）

各項目にスコア・根拠・タイムスタンプ・改善トーク例が付き、履歴とダッシュボード（②ヒアリング深度×成約率など）も見られます。

## 使い方

1. [公開ページ](https://taketo0904.github.io/taishoku-smile-ai-scoring/) を開く
2. 「設定」タブでAI（Gemini無料枠 または Claude）のAPIキーを入力
3. 「採点する」タブで商談の文字起こしを貼り付け、または音声/動画ファイルをアップロード
4. 「AIで採点する」を押す

## データについて

単体HTMLファイルで、サーバーは持ちません。APIキー・採点履歴・文字起こしはすべてブラウザのlocalStorageに保存され、外部には送信されません（採点時のみ、入力した文字起こしが選択したAI（GoogleまたはAnthropic）のAPIに送信されます）。

localStorageはブラウザ・端末ごとに独立しているため、履歴・ダッシュボードは個人ごとの記録になります。チーム全体の集計が必要な場合は別途設計が必要です。

## 開発

単体HTML（`index.html`）です。ビルド不要、GitHub Pagesでそのまま配信しています。
